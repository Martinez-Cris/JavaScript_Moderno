JavaScript y JavaScript Moderno (ES6+)

JavaScript (Tradicional)

JavaScript es un lenguaje de programación que se utiliza principalmente para crear interactividad en las páginas web. Es un lenguaje de scripting que se ejecuta en el navegador del usuario, lo que permite a los desarrolladores crear experiencias dinámicas y atractivas. Es la versión más antigua del lenguaje (antes de 2015). Tiene una sintaxis más larga y limitada.

JavaScript Moderno

JavaScript moderno es simplemente la forma actual de escribir JavaScript. Una actualización importante del lenguaje lanzada en 2015 (también llamada ECMAScript 6). Trajo nuevas formas más simples, claras y potentes de escribir código.

Promesas (Promises)

Las promesas son un mecanismo para manejar operaciones asíncronas en JavaScript. Permiten gestionar mejor las operaciones que pueden tardar en completarse.
En palabras simples, es una forma de manejar tareas que tardan tiempo (como cargar datos), indicando qué hacer cuando terminan.
Representa la eventual finalización (o falla) de una operación asíncrona y su valor resultante, cumplida o rechazada. Esto facilita la escritura de código asíncrono de manera más legible y manejable.
