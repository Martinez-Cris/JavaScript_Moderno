JavaScript Moderno (ES6+) y Angular

JavaScript Moderno

JavaScript moderno es simplemente la forma actual de escribir JavaScript, utilizando las características nuevas que se han ido agregando al lenguaje en los últimos años.

Antes (JavaScript antiguo) se escribía algo así:

Ahora con JavaScript moderno se escribe así:

const nombre = "Cristian";

const saludar = () => {
console.log(`Hola ${nombre}`);
};

Características comunes del JavaScript moderno:

  • let y const en lugar de var
  • Arrow functions ()=>
  • Template strings `Hola ${nombre}`
  • Modules (import / export)
  • Async / await
  • Clases

ES6 (ECMAScript 2015)

ES6 significa ECMAScript 6, también llamado ECMAScript 2015.

ES6 fue una actualización muy grande que introdujo muchas cosas nuevas como:

  • let y const
  • Arrow functions ()=>
  • Clases
  • Módulos import / export
  • Promesas Promise
  • Destructuring
  • Spread operator ...
